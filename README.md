# GSAP Landing Page

A modern responsive landing page built with React, Vite, GSAP, and CSS.

## Features

- Responsive layout
- GSAP animations
- Hero section with scroll animations
- Fullscreen animated menu
- Smooth hover effects
- Modern UI
- Component-based React structure

## Tech Stack

- React
- Vite
- GSAP
- CSS3
- JavaScript (ES6)
